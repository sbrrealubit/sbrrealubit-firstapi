# Student REST API

A simple REST API built using Express.js and MySQL.

## Features
- Get all students
- Get single student
- Add student
- Update student
- Update student status
- Delete student

## Technologies Used
- Node.js
- Express.js
- MySQL (mysql2)
- dotenv
- cors

## Database
- Database name: db_realubit
- Table: tbl_student
- SQL file included in repository

## How to Run
1. Install dependencies  
   npm install

2. Configure `.env` file

3. Start server  
   node index.js

Server runs on: http://localhost:3000
